# siteMap

## Compile files
javac JavaWebCrawlerRun.java
or
javac *.java
## Run
java JavaWebCrawlerRun -p project-name -u url -d domain-name

### Project Name Format

Any directory standard string

### Url Format

http(s)://domain-name
e.g https://www.zoho.com

### Domain Name

e.g. zoho.com

#### Thread count set to 2 by default
